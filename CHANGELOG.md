# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),

## [1.0.0] - 2025-11-23

### Added
- Initial release of the Personal Link Hub.
- Dynamic metadata generation from a central `data.ts` file.
- Favicon dynamically loaded from `data.ts`.
- SEO-ready with `robots.txt` and `sitemap.xml`.
- Styled with Tailwind CSS, `shadcn/ui`, and Geist font.
- Comprehensive `README.md` with setup and deployment instructions.
- MIT License and `CHANGELOG.md`.
